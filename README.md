## 站点简介

这是一个面向程序员的**个人知识库**，旨在整理软件开发中的核心知识点和常用工具。

很多时候，重要的工具和知识点因为平时使用频率不高而容易遗忘。当需要解决特定问题时，四处查找资料非常浪费时间。本站点的目标是系统性地整理开发中用到的各种基础知识和工具，方便快速查阅。
