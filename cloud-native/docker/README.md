# Docker

Docker 是一个开源的应用容器引擎，让开发者可以打包他们的应用以及依赖包到一个可移植的容器中。

## 核心概念

- 镜像（Image）：应用的只读模板
- 容器（Container）：镜像的运行实例
- 仓库（Registry）：镜像存储服务
- Dockerfile：构建镜像的脚本
