# Kubernetes

Kubernetes (K8s) 是用于自动化部署、扩展和管理容器化应用程序的开源系统。

## 核心概念

- Pod：最小部署单元
- Service：服务发现与负载均衡
- Deployment：声明式应用部署
- ConfigMap & Secret：配置与敏感信息管理
- Ingress：HTTP/HTTPS 路由
